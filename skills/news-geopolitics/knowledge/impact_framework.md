# News Geopolitics - 核心规则

## 职责边界

### ✅ 只做这些
- 抓取国内外重大新闻
- 事件去重聚类
- 给事件评级
- 判断影响哪些资产/行业/地区
- 输出短中期影响

### ❌ 不做这些
- 不直接下投资指令
- 不把未经证实消息当事实
- 不做情绪化判断

---

## 事件分类

| 分类 | 示例 |
|------|------|
| 金融市场 | 股市崩盘、金融危机 |
| 地缘冲突 | 战争、制裁 |
| 能源 | 原油、天然气 |
| 科技 | 芯片禁令、AI限制 |
| 国际贸易 | 关税、贸易战 |
| 国内政策 | 刺激政策、降准 |
| 自然灾害 | 疫情、台风 |

---

## 影响映射表

| 事件 | 影响资产 |
|------|----------|
| 地缘冲突升级 | 黄金、原油、航运、军工 |
| 贸易摩擦 | 出口、汇率、制造业 |
| 美联储偏鹰 | 美债、美元、科技股 |
| 国内刺激 | 地产链、消费、金融 |
| 黑天鹅 | 流动性优先、降低高波动 |

## 风控规则

- 未证实消息标注"待确认"
- 多源交叉提高置信度
- 传闻和确认新闻分级处理
